Установка русского языка на Opencart 3.x:
1. Установить архив russian_oc3.ocmod.zip через админку или распаковать архив и загрузить файлы на сервер в соответствующие папки сайта
2. В админпанели открыть раздел System > Localisation > Languages > + (Add new) и пропишите в поля:
 - Language Name: Русский
 - Code: ru-ru
 - Locale: ru-RU,ru_RU.UTF-8,ru_RU,ru-ru,russian
 - Status: Enabled
 - Sort Order: 1
3. В админпанели перейдите в раздел System > Settings > Edit > вкладка Local:
 - выберите Language: Русский
 - выберите Administration Language: Русский
